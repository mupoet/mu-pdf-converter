不只是格式转换——高保真还原、智能解析、AI增强，三者协同工作。

🔄

四层叠加 PPT

位图 + SVG矢量 + 原生表格 + 可编辑文本框。打开PPT后文字可选、表格可增行、图形可缩放。

📊

三引擎表格提取

XY-Cut → MarkItDown → pdfplumber 三级降级。无边框、合并单元格、不规则布局——三重保险。

🌍

外文自动翻译

自动检测外文PDF，翻译同时保留排版。双引擎（translators + 本地LLM）加专有名词保护。

📝

双路径表单填写

路径A：标准AcroForm字段。路径B：AI分析扁平PDF → 坐标注释。覆盖所有表单类型。

🧹

智能去水印

4种策略：文字 / 图片 / XObject / 激进模式。支持预览确认（--detect-only）再执行。

🔌

MCP Server

内置 JSON-RPC 2.0（stdin/stdout）。零外部依赖。任何MCP兼容的AI助手可直接调用。

🏗️ 技术架构
